## 标准化说明

NamuWiki 英文镜像中的部分活动名为机器翻译，本项目统一转换为站内使用的 Daily Challenge 名称：

| NamuWiki 文案 | 站内名称 |
| --- | --- |
| Personal employment | Private Hire |
| Commuting time | Rush Hour |
| E-PAY blackout | E-Payment Outage |
| City trip | Urban Odyssey |
| Marathon finish | Marathon Road Closure |
| Marathon preparation | Marathon Shuttle |
| Leg closure | Bridge Closure |
| Street power outage | Street Light Outage |
| Rare, personal employment | Rare Appearance x Private Hire |
| Rare car / rare | Rare Appearance |
| Kindly/Friendly passenger | Friendly Day |
| Lazy passenger | Lazy Passengers |
| Slow passenger | Slow Passengers |
| Sponsorship | Sponsored Route |
| Concert | Concert Aftermath |
| Day of complaint | Grumble Day |

## 备注

- 这些是历史资料，不会影响 `getTodaysDailyChallenge()` 的今日挑战，除非当前游戏日真的落在 2024 年。
- 日历弹窗可通过年份/月选择器查看这些月份；点击有活动的日期会打开对应 Daily Challenge 详情。
